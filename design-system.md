# Design System — МегаСитиШоп Портал Обучения

## Бренд
МегаСитиШоп — мебель для образования (школы, ДОУ, нац. проекты).
Маскот: Семёновна (сова в бирюзовом пиджаке).
Стиль: дружелюбный, яркий, образовательный — не строгий корпоратив.

---

## Цвета

| Token          | Hex       | Использование                              |
|----------------|-----------|---------------------------------------------|
| `--teal`       | `#2ABFC0` | Основной акцент, кнопки, иконки, прогресс   |
| `--teal-dark`  | `#1A9EA0` | Hover-состояния, активные элементы          |
| `--teal-light` | `#A8E8E8` | Светлые фоны, бейджи, подсветка             |
| `--cream`      | `#F5E4C0` | Вторичные фоны, карточки, секции            |
| `--black`      | `#0D0D0D` | Основной текст, заголовки                   |
| `--gray-1`     | `#F4F4F4` | Светлые фоны страниц                        |
| `--gray-2`     | `#B0B0B0` | Вторичный текст, borders                    |
| `--white`      | `#ffffff` | Карточки, фоны блоков                       |
| `--success`    | `#4CAF50` | Тест пройден, галочки                       |
| `--error`      | `#F44336` | Неправильный ответ                          |

### Правила использования
- Светлый портал: белый/серый фон + `#0D0D0D` текст + `#2ABFC0` акцент
- Тёмные блоки: `#0D0D0D` фон + белый текст + `#2ABFC0` акцент
- Никогда не использовать `--teal` на светло-бирюзовом фоне

## Типографика

### Основной шрифт: Inter
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700;800&display=swap" rel="stylesheet">
```
Чистый, читаемый, современный — идеален для обучающих платформ.
Fallback: `'Helvetica Neue', Arial, sans-serif`

### Шкала размеров
| Роль        | Размер  | Вес  | Примечание              |
|-------------|---------|------|--------------------------|
| H1          | 40px    | 800  | Заголовки страниц        |
| H2          | 28px    | 700  | Заголовки секций         |
| H3          | 20px    | 600  | Карточки, уроки          |
| Label       | 11px    | 600  | UPPERCASE, letter-spacing: 2px |
| Body        | 16px    | 400  | Основной текст           |
| Caption     | 13px    | 400  | Подписи, мета            |

## Компоненты

### Кнопки
```css
/* Primary */
background: #2ABFC0; color: #ffffff; font-weight: 600;
padding: 14px 32px; border-radius: 8px;

/* Secondary */
background: transparent; border: 2px solid #2ABFC0; color: #2ABFC0;
border-radius: 8px;

/* Disabled */
background: #B0B0B0; color: #ffffff; cursor: not-allowed;
```

### Карточки уроков
```css
background: #ffffff; border: 1px solid #E8E8E8;
border-radius: 12px; padding: 24px;
box-shadow: 0 2px 8px rgba(0,0,0,0.06);
```

### Прогресс-бар
```css
background: #E8E8E8; border-radius: 4px; height: 8px;
fill: #2ABFC0;
```

### Бейдж статуса
```css
/* Пройдено */
background: #E8F5E9; color: #2E7D32; border-radius: 20px; padding: 4px 12px;

/* В процессе */
background: #E0F7FA; color: #00838F; border-radius: 20px; padding: 4px 12px;

/* Заблокировано */
background: #F4F4F4; color: #B0B0B0; border-radius: 20px; padding: 4px 12px;
```

## Отступы и сетка

- Base unit: `8px`
- Значения: 8, 16, 24, 32, 48, 64px
- Максимальная ширина контента: `960px`
- Border-radius карточек: `12px`
- Border-radius кнопок: `8px`

---

## Принципы дизайна

1. **Дружелюбный** — не строгий корпоратив, светло и тепло
2. **Чёткий прогресс** — менеджер всегда видит где он и сколько осталось
3. **Мобильный** — портал открывается на телефоне без проблем
4. **Акцент на действие** — одна главная кнопка на экране
5. **Семёновна** — маскот появляется в ключевых моментах (приветствие, поздравление с прохождением теста)
